# JobHive – Modern Job Board

A responsive Job Board application built with React and Vite.

## Features

- Responsive UI
- Modern Hero Section
- Search Jobs
- Job Listings
- Job Details Page
- Responsive Navigation
- GitHub Actions CI
- Vercel Deployment

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- React Icons

## Installation

```bash
git clone <repository-url>

cd job-board

npm install

npm run dev
```

## Build

```bash
npm run build
```
